**On this page**, **how** will you introduce your worklog? In **how many weeks** did you complete the program? **What did you do** during those weeks?

Typically and as a standard, a worklog is carried out over about 3 months (during the internship period) with the weekly content as follows:

**Week 1:** [Getting started with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Deploying EC2, practicing IAM Roles and advanced AWS CLI](1.2-week2/)

**Week 3:** [Mastering VPC, Subnet, Internet Gateway, and SG/NACL security](1.3-week3/)

**Week 4:** [Leveraging Amazon S3: Storage Classes, Versioning, and Static Website Hosting](1.4-week4/)

**Week 5:** [Deploying Amazon RDS, configuring Multi-AZ and Read Replicas](1.5-week5/)

**Week 6:** [Working with DynamoDB (NoSQL) and ElastiCache (Caching Layer)](1.6-week6/)

**Week 7:** [Configuring Auto Scaling Group (ASG), ALB, and monitoring with CloudWatch](1.7-week7/)

**Week 8:** [Building Serverless architecture with AWS Lambda and API Gateway](1.8-week8/)

**Week 9:** [Processing Data Streaming with Amazon Kinesis and ETL process with AWS Glue](1.9-week9/)

**Week 10:** [Integrating asynchronous applications using SQS, SNS, and Step Functions](1.10-week10/)

**Week 11:** [Deploying advanced security: AWS WAF, Shield, and GuardDuty](1.11-week11/)

**Week 12:** [Summarizing all knowledge, cleaning up resources, and certification orientation](1.12-week12/)
